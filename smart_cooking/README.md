# 🍳 Smart Cooking App

## 📌 Description
Smart Cooking App that calculates ingredient quantities based on number of people and provides recipe guidance with a user-friendly interface.

---

## 🚀 Features
- Ingredient quantity calculation
- Recipe suggestions
- User-friendly interface
- Supports dynamic input (adults/children)

---

## 🏗️ Project Structure
- Android App (Frontend)
- FastAPI Backend (Server)

---

## 🛠️ Technologies Used
- Java (Android)
- XML (UI Design)
- Python (FastAPI)
